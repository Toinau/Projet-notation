Instructions
Format du fichier CSV :

Votre fichier doit contenir les colonnes suivantes (séparées par des points-virgules) :

Colonne Obligatoire Description
prenom Oui Prénom de l'utilisateur
nom Oui Nom de famille
email Oui Adresse email (unique)
telephone Recommandé Numéro de téléphone
role Non coureur ou admin
equipe Non Nom de l'équipe
notifications_sms Non true/false
Exemple de contenu :
prenom;nom;email;telephone;role;equipe;notifications_sms
Jean;Dupont;jean.dupont@email.com;0123456789;coureur;Équipe A;true
Marie;Martin;marie.martin@email.com;0987654321;admin;;false
Conseil : Utilisez le bouton "Télécharger Template" pour obtenir un fichier d'exemple avec la structure correcte.
